## 🔧 Comandos Útiles

### **Tailwind CSS**
```bash
# Instalar Tailwind
npm install -D tailwindcss

# Generar config
npx tailwindcss init

# Build CSS
npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch

```

## 📝 Cheat Sheets

### **CSS Grid Cheat Sheet**
```css
/* Container */
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: auto;
  gap: 1rem;
  grid-template-areas: 
    "header header header"
    "sidebar main main"
    "footer footer footer";
}

/* Items */
.grid-item {
  grid-column: 1 / 3;
  grid-row: 1 / 2;
  grid-area: header;
}
```

### **Flexbox Cheat Sheet**
```css
/* Container */
.flex-container {
  display: flex;
  flex-direction: row;
  justify-content: center;
  align-items: center;
  flex-wrap: wrap;
  gap: 1rem;
}

/* Items */
.flex-item {
  flex: 1 1 200px;
  align-self: flex-start;
}
```

### **Media Queries Cheat Sheet**
```css
/* Mobile First */
.element {
  /* Mobile styles */
}

@media (min-width: 640px) {
  .element {
    /* Tablet styles */
  }
}

@media (min-width: 1024px) {
  .element {
    /* Desktop styles */
  }
}
```
